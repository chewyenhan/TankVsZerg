# Tank vs Zerg — Side-Scrolling Defense

A 2-player local competitive arcade shooter built with Phaser.js. Two tanks defend against waves of Zerg enemies while trying to destroy each other. Best-of-3 rounds.

## Controls

### Player 1 (Red Tank)
| Key | Action |
|-----|--------|
| W/A/S/D | Move |
| Space | Fire main cannon |
| Shift | Burst fire (uses shield) |
| E | Shield boost |

### Player 2 (Blue Tank)
| Key | Action |
|-----|--------|
| Arrow Keys | Move |
| Enter | Fire main cannon |
| Right Shift | Burst fire |
| I | Shield boost |

### Shared
| Key | Action |
|-----|--------|
| Escape | Pause / Resume |

## Play

Open `index.html` in a browser. No server needed.

## Tech

- **Phaser.js 3.87** via CDN
- **Arcade Physics** — top-down 2D, no gravity
- **Procedural Graphics** — all sprites drawn with Phaser Graphics API
- **Web Audio** — oscillator-based sound effects
